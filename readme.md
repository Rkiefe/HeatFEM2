# HeatFEM
This is a finite element implementation of the 2D heat equation, with Neumann boundary conditions.

## Heat Equation
$\rho C_p \frac{\partial T}{\partial t} - \frac{1}{C_p}\nabla \cdot (k\nabla T) = q_V$
